# Web components
Standard components for all frameworks and all platforms.
### Installation

```sh
yarn add @uit/button
```

### Usage
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <title>Web component example</title>
  </head>
  <body>
    <uit-button>My button</uit-button>
    <script type="module">
      import '@uit/button';
    </script>
  </body>
</html>
```
